По клику  "Создать бриф", OpenAI API формирует простой текстовый бриф для дизайнера на основе тренда.
При нажатии:

Берутся данные по тренду:
	- название тренда;
	- тип;
	- статистика роста;
	- последние 10–20 постов;
	- связанные сущности.
	
В отдельном окне отображается простой текстовый промпт. Вот пример кода, который должен его формировать:
	# brief_generator.py

	def build_prompt(trend, examples):
		return f"""
	Ты арт-директор бренда дизайнерских футболок.

	Тренд:
	{trend.name}

	Тип:
	{trend.entity_type}

	Рост:
	{trend.growth_7d}%

	Примеры упоминаний:
	{examples}

	Сформируй:
	1. Описание тренда
	2. Целевую аудиторию
	3. 5 идей для принтов
	4. Рекомендации по стилю
	5. Риск нарушения авторских прав
	"""
