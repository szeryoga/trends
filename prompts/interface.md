Главная `/`

* Поле для ввода: "Количество постов: N" (по умолчанию N=10)
* Кнопка "Собрать посты". По клику происходит сбор последних N постов из активных каналов.

* Таблица:
    * топ-20 трендов за последние 7 дней;
    * фильтр по типу сущности;
    * фильтр по категории канала;
    * сортировка 

    Пример:
    | Тренд      |  Рост | Каналов | Действие     |
    | ---------- | ----: | ------: | ------------ |
    | Y2K        | +230% |      12 | Создать бриф |
    | Белые ночи | +180% |       8 | Создать бриф |
    | Ретро ПК   | +160% |       6 | Создать бриф |


    Приложение считает не “рост постов”, а рост упоминаний сущности/тренда.
    Например, тренд Y2K:
        последние 7 дней: 35 упоминаний
        предыдущие 7 дней: 10 упоминаний

    Формула:
        growth_7d = (35 - 10) / 10 * 100 = +250%

    Если раньше было 0 упоминаний, лучше ставить специальный статус:
        new_trend = true
        growth_7d = null


    * По клику кнопки "Создать бриф": см create_brief.md
	


Страница тренда `/trends/{entity}`

	* динамика упоминаний по дням (график роста по дням);
	* список каналов, где встречалась сущность;
	* примеры постов;
	* тип сущности;
	* источники извлечения;
	* примерный дизайн-потенциал.

Страница постов `/posts`

	* последние собранные посты;
	* фильтр по каналу;
	* фильтр по найденной сущности;
	* отображение найденных entities под каждым постом.


Страница каналов `/channels`

	* список каналов. По умолчанию:
        Лентач
        МДК
        Двач
        Код Дурова
        Технокотики
        Бумага
        Ротонда
        Афиша Daily
        The Flow
        Design Hunters
	* форма добавления канала;
	* для кажного канала: включение/отключение мониторинга.

Страница настройка `/settings`
    настройка ежедневного запуска сбора постов по расписанию.
